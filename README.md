# Full-Stack Streaming Application

**Tech Stack**: Next.js, Typescript, Stream io, Tailwind CSS, Clerk js.

### App Features:

- Developed a full-featured video streaming platform enabling users to create, host, and join live video sessions.
- Implemented secure authentication and user management using Clerk.js, supporting both registered and guest users.
- Integrated Stream.io for seamless real-time video streaming, chat, and media synchronization, ensuring high-quality, low-latency performance.
- Leveraged Next.js (App Router) and TypeScript for a scalable, maintainable, and performant full-stack architecture.
- Implemented session recording and playback features for on-demand content access.
- Focused on clean, reusable components and server-side rendering (SSR) to improve SEO and initial load performance.
